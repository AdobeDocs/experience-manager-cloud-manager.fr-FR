# Notes de mise à jour de la version 2019.7.0 de Cloud Manager {#release-notes-for}

La version  2019.7.0 ajoute des mises à jour aux notifications d’Experience Cloud et des améliorations en tant que correctifs. Pour plus d’informations, consultez les sections ci-après.

## Date de publication {#release-date}

La date de publication de la version 2019.7.0 de [!UICONTROL Cloud Manager] est le vendredi 18 juillet 2019.

## Nouveautés {#whats-new}

Une notification Experience Cloud est maintenant envoyée au début d’un déploiement en production.

## Correctifs {#bug-fixes}

* Dans certains cas, Cloud Manager effectuait une analyse de code statique sur les fichiers Python et PHP.
* Les packages contenant des FileVault InstallHooks n’étaient pas transmis de manière cohérente à l’étape Qualité du code.
* Dans certaines combinaisons, les problèmes de qualité du code n’étaient pas systématiquement triés.
* La page d’exécution du pipeline présentait certains problèmes visuels.
* L’étape de test de performance peut échouer de manière aléatoire, parfois en raison de contraintes de ressources liées à l’infrastructure cloud sous-jacente.
* Certaines compilations de clients échouaient en raison de problèmes de réseau.
