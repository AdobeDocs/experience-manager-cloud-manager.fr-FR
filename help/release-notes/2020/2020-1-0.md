# Notes de mise à jour de la version 2020.1.0 de Cloud Manager {#release-notes-for}

La section suivante présente les notes de mise à jour générales de  2020.1.0 et ajoute des mises à jour relatives à l’accès aux informations d’identification Git et à l’expérience de connexion.

## Date de publication {#release-date}

La date de publication de la version 2020.1.0 de [!UICONTROL Cloud Manager] est le vendredi 16 janvier 2020.

## Nouveautés {#whats-new}

* Vous pouvez désormais obtenir des informations d’identification Git à partir de l’interface utilisateur de Cloud Manager. Voir [Accès à Git](/help/managing-code/managing-repositories.md) pour plus d’informations.
* L’expérience de connexion et la structure de l’URL ont été modifiées dans le cadre d’une initiative à l’échelle d’Adobe. Les anciens signets redirigent vers les nouvelles URL.


## Correctifs {#bug-fixes}

* Les déploiements sur des topologies de création uniquement n’ont pas déployé les modifications de configuration de Dispatcher.
* Dans certaines configurations, la création d’un pipeline de type Qualité de code uniquement s’avérait impossible.
* La carte de résumé de l’environnement sur la page d’aperçu ne s’affichait pas toujours correctement.
* Les exécutions de pipeline pouvaient expirer sur les topologies volumineuses.
