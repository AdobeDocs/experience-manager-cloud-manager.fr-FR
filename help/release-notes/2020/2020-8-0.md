# Notes de mise à jour de la version 2020.8.0 {#release-notes-for}

La section ci-dessous présente les notes générales de mise à jour de la version 2020.8.0 de [!UICONTROL Cloud Manager].

## Date de publication {#release-date}

La date de publication de la version 2020.8.0 de [!UICONTROL Cloud Manager] est le vendredi 6 août 2020.

## Nouveautés {#whats-new}

Les référentiels Maven privés liés à l’authentification sont désormais pris en charge.

## Correctifs {#bug-fixes}

* Certains plug-ins SonarQube inutiles et indésirables étaient en cours d’exécution dans le cadre de l’analyse de la qualité du code.

* Sur la page d’exécution du pipeline, le nom de la branche n’était pas correctement formaté.

* Lors d’un déploiement sur des topologies avec une seule publication, un seul Dispatcher et un auteur Cold Standby, le Dispatcher a été supprimé par erreur de la répartition de charge.

* Dans certains cas, les exécutions de pipeline terminées n’étaient pas correctement enregistrées comme étant terminées. Ce problème a empêché de nouvelles exécutions du pipeline.

* Les exécutions de pipelines étaient parfois *bloquées* en raison de problèmes de communication interne.

* Les info-bulles des cartes de programme n’étaient pas toujours correctes.

* Les couleurs n’étaient pas cohérentes sur la page **Aperçu**.

* Les tests de performances des sites prennent désormais en charge l’utilisation facultative de l’authentification.

* Les caches Dispatcher des instances de création sont automatiquement vidés lorsque les configurations Dispatcher sont déployées via Cloud Manager.
