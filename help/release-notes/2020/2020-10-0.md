# Notes de mise à jour de la version 2020.10.0 de Cloud Manager {#release-notes-for}

La section ci-dessous présente les notes générales de mise à jour de la version 2020.10.0 de [!UICONTROL Cloud Manager].

## Date de publication {#release-date}

La date de publication de la mise à jour 2020.10.0 de [!UICONTROL Cloud Manager] est le vendredi 1 octobre 2020.

## Correctifs {#bug-fixes}

* Le moteur de recherche utilisé pour les tests de performances prenait en compte de manière incorrecte certains types de ressources comme des liens web valides.

* Dans certains cas, l’étape d’achèvement dans les tests de performance n’était pas correctement gérée, ce qui entraînait des étapes de longue durée.

* Lorsque l’invalidation du cache de Dispatcher était configurée pour les déploiements d’exploitation, elle était parfois exécutée deux fois.
